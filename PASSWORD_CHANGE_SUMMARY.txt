✅ تم إضافة ميزة تغيير كلمة المرور من قبل الإدارة

═══════════════════════════════════════════════════════════════════════

🎯 الميزة الجديدة:

يمكن للمسؤول (Admin) الآن:
  ✨ إنشاء كلمة مرور مؤقتة عشوائية للمستخدم
  ✨ تعيين كلمة مرور محددة للمستخدم
  ✨ نسخ الكلمة ارسالها للمستخدم

═══════════════════════════════════════════════════════════════════════

🚀 كيفية الاستخدام:

1. اذهب إلى: Admin Panel → User Management
2. ابحث عن المستخدم
3. اضغط الثلاث نقاط (⋯) بجانب المستخدم
4. اختر: "تغيير كلمة المرور" 🔐
5. اختر خياراً:
   ✓ إنشاء كلمة مرور مؤقتة (عشوائية آمنة)
   ✓ تعيين كلمة مرور محددة (أنت تختار)

═══════════════════════════════════════════════════════════════════════

🎯 الخيارات المتاحة:

1️⃣ إنشاء كلمة مرور مؤقتة:
   - نقر زر واحد
   - الخادم ينشئ كلمة عشوائية آمنة
   - تنسخ وترسل للمستخدم
   - مثال: Temp5847293

2️⃣ تعيين كلمة مرور محددة:
   - أدخل كلمة المرور (6+ أحرف)
   - أعد إدخالها للتأكيد
   - تحديث فوري

═══════════════════════════════════════════════════════════════════════

📝 الحالات الاستخدام:

✓ مستخدم نسي كلمة المرور → إنشاء مؤقتة
✓ مستخدم جديد → إنشاء مؤقتة
✓ كلمة مرور ضعيفة → تعيين محددة
✓ تحديث دوري للأمان → أي خيار

═══════════════════════════════════════════════════════════════════════

✅ اختبر الميزة:

1. قم بتشغيل التطبيق كالعادة
2. سجل دخول كـ Admin
3. اذهب إلى Admin Panel → User Management
4. اختر أي مستخدم
5. اضغط الثلاث نقاط → "تغيير كلمة المرور"
6. جرب الخيارات!
