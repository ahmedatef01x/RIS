### 5️⃣ **تحديث حالة الدفع الفوري**

عند تحديث الحالة:

1. **الواجهة:**
```tsx
<Button onClick={() => handlePaymentStatusChange("paid")}>
  تحديث: مدفوع
</Button>
```

2. **العملية:**
- ✅ جلب حالة الدفع الحالية
- ✅ عرض أزرار التحديث (مفعل / معطل)
- ✅ تحديث قاعدة البيانات عند النقر
- ✅ تسجيل وقت الدفع تلقائياً
- ✅ عرض رسالة تأكيد
- ✅ تحديث الواجهة فوراً

3. **البيانات المحدثة:**
```sql
UPDATE billing
SET 
  status = 'paid',
  paid_at = GETDATE(),
  notes = @notes
WHERE id = @billing_id
```

## 🗄️ جدول البيانات المستخدم

### جدول `billing`:
```sql
billing (
  id (PK)
  patient_id (FK)
  exam_order_id (FK)
  amount_egp DECIMAL(10,2)
  discount_egp DECIMAL(10,2)
  insurance_coverage_egp DECIMAL(10,2)
  total_due_egp (COMPUTED)
  status NVARCHAR(50)          ← مدفوع / معلق
  payment_method NVARCHAR(50)
  insurance_approval NVARCHAR(255)
  notes NVARCHAR(MAX)
  paid_at DATETIME2
  created_by (FK)
  created_at DATETIME2
)
```
