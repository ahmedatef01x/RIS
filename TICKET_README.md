# 🎫 نظام تذكرة الفحص والدفع المتقدم

## 🌟 نظرة عامة

تم تطوير نظام متكامل لطباعة وإدارة تذاكر الفحوصات الطبية مع إمكانية عرض وتحديث حالة الدفع مباشرة من قائمة العمل.

---

## 🎯 الميزات الرئيسية

### 📋 1. تذكرة فحص متقدمة
```
نافذة احترافية تعرض:
✅ معلومات المريض (الاسم، الرقم)
✅ معلومات الفحص (النوع، الجهاز، الموعد)
✅ معلومات الدفع (الحالة، المبلغ)
✅ ملاحظات إضافية
✅ أزرار تحديث الحالة
✅ زر طباعة احترافي
```

### 💳 2. عمود حالة الدفع
```
في جدول قائمة العمل:
✅ عرض حالة الدفع (مدفوع/معلق)
✅ عرض المبلغ المستحق
✅ تحديث فوري من قاعدة البيانات
```

### 🔄 3. تحديث الدفع الفوري
```
عند تحديث الحالة:
✅ حفظ فوري في قاعدة البيانات
✅ تسجيل وقت الدفع تلقائياً
✅ تحديث الواجهة تلقائياً
✅ رسائل تأكيد واضحة
```

### 🖨️ 4. طباعة احترافية
```
تذكرة بتصميم عالي الجودة:
✅ شعار وعنوان المركز
✅ بيانات المريض والفحص
✅ رقم الزيارة والتاريخ
✅ حالة الدفع والمبلغ
✅ تنسيق عربي RTL محسّن
✅ صالحة للطباعة على الورق
```

## 📱 كيفية الاستخدام

### الخطوة 1: فتح قائمة العمل
```
القائمة الجانبية → قائمة العمل (Worklist)
```

### الخطوة 2: البحث عن الفحص
```
استخدم شريط البحث للبحث عن:
• اسم المريض
• رقم المريض
• نوع الفحص
```

### الخطوة 3: فتح التذكرة
```
النقر على (⋯) ← اختر "طباعة تذكرة"
```

### الخطوة 4: تحديث حالة الدفع (اختياري)
```
في قسم "حالة الدفع":
✓ اضغط "تحديث: مدفوع" إذا قام المريض بالدفع
✓ اضغط "تحديث: معلق" إذا لم يدفع بعد
```

### الخطوة 5: طباعة
```
اضغط [🖨️ طباعة التذكرة]
← ستُطبع مباشرة أو تظهر نافذة طباعة
```

## 📞 الدعم والمساعدة

### مشاكل شائعة وحلولها

#### المشكلة: "لا توجد فحوصات"
```
الحل:
1. جرّب: node seed-test-exams.js
2. أعد تحميل الصفحة (F5)
3. تحقق من اتصال قاعدة البيانات
```

#### المشكلة: "التذكرة فارغة"
```
الحل:
1. البيانات الناقصة طبيعي أن تكون فارغة
2. تحقق من بيانات الفحص
3. عدّل البيانات المفقودة
```

#### المشكلة: "عدم حفظ التحديثات"
```
الحل:
1. تأكد من تشغيل الخادم
2. افتح Developer Tools (F12)
3. تحقق من رسائل الخطأ في Console
4. أعد التحميل
```
