## 📋 استخدام التذكرة

### السيناريو 1️⃣: طباعة تذكرة فحص

1. **الذهاب لقائمة العمل**
   ```
   القائمة الجانبية → قائمة العمل (Worklist)
   ```

2. **البحث عن الفحص**
   ```
   استخدم شريط البحث للبحث عن:
   • اسم المريض
   • رقم المريض (MRN)
   • نوع الفحص
   ```

3. **فتح التذكرة**
   ```
   النقر على (⋯) ← اختر "طباعة تذكرة"
   ```

4. **عرض التذكرة**
   تظهر نافذة تعرض:
   - ✅ معلومات المريض
   - ✅ معلومات الفحص
   - ✅ حالة الدفع والمبلغ
   - ✅ أزرار الطباعة والإغلاق

5. **طباعة**
   ```
   اضغط [🖨️ طباعة التذكرة]
   ← ستُطبع مباشرة أو تظهر نافذة طباعة
   ```

---

### السيناريو 2️⃣: تحديث حالة الدفع

1. **فتح التذكرة** (نفس الخطوات أعلاه)

2. **تحديد حالة الدفع**
   ```
   قسم "حالة الدفع":
   
   اختر:
   • [✓ تحديث: مدفوع] ← إذا دفع المريض
   • [⏳ تحديث: معلق] ← إذا لم يدفع بعد
   ```

3. **التأكيد**
   - تظهر رسالة خضراء: "تم تحديث الحالة إلى..."
   - يتم حفظ التحديث تلقائياً

4. **إغلاق النافذة**
   ```
   اضغط [إغلاق]
   ← ستُحدّث قائمة العمل تلقائياً
   ```

---

### السيناريو 3️⃣: إضافة ملاحظات

1. **في نافذة التذكرة**
   ```
   قسم "ملاحظات" ← اكتب أي معلومات إضافية
   ```

2. **أمثلة على ملاحظات:**
   - "المريض لديه تحفظ معادن"
   - "يرجى الصيام قبل الفحص"
   - "تم دفع 300 والباقي لاحقاً"

3. **الملاحظات ستظهر في:**
   - نافذة التذكرة
   - التذكرة الطباعية

## 💡 نصائح مهمة

### 1. البيانات الناقصة
```
• إذا لم تكن هناك بيانات ستظهر: —
• لا تقلق، هذا طبيعي إذا لم تُكمل البيانات
```

### 2. التحديثات التلقائية
```
• عند تحديث حالة الدفع:
  ← تُحدّث قاعدة البيانات فوراً
  ← تُحدّث الجدول تلقائياً
  ← لا حاجة لإعادة تحميل
```

### 3. الطباعة
```
• يمكنك:
  ✓ طباعة مباشرة على الطابعة
  ✓ حفظ كـ PDF
  ✓ عرض قبل الطباعة
```

### 4. البحث والتصفية
```
جدول العمل يحتوي على:
✓ بحث بالاسم
✓ بحث برقم المريض
✓ بحث بنوع الفحص
✓ تصفية بالحالة
✓ تصفية بالجهاز
```

## ⚠️ المشاكل الشائعة

### المشكلة 1: "لا توجد فحوصات"
```
الحل:
1. تأكد من إضافة فحوصات للمريض
2. جرّب: node seed-test-exams.js
3. أعد تحميل الصفحة (F5)
```

### المشكلة 2: "التذكرة فارغة"
```
الحل:
1. تأكد من أن الفحص لديه بيانات كاملة
2. اترك البيانات الناقصة فارغة (طبيعي)
3. تحقق من اتصال قاعدة البيانات
```

### المشكلة 3: "عدم حفظ التحديثات"
```
الحل:
1. تأكد من تشغيل الخادم المحلي
2. تحقق من رسالة الخطأ
3. أعد تحميل الصفحة
```

## 🎓 خطوات التعلم

**المبتدئ:**
1. افتح قائمة العمل
2. جرّب البحث عن فحص
3. اضغط "طباعة تذكرة"

**المتقدم:**
1. عدّل حالات الدفع
2. أضف ملاحظات مخصصة
3. اطبع تذاكر متعددة

**الخبير:**
1. استخدم البحث والتصفية المتقدم
2. تتبع حالات الدفع تلقائياً
3. طباعة دفعات من التذاكر

## 📞 الدعم

إذا واجهت مشاكل:
1. تحقق من الأخطاء في console (F12)
2. تأكد من تشغيل الخوادم
3. أعد تحميل الصفحة
4. جرّب تصفير الـ cache (Ctrl+Shift+Delete)
