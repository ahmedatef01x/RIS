### 2. **تحديث صفحة Worklist** 📋

**التعديلات:**
```tsx
// 1. إضافة الاستيراد
import { ExamTicket } from "@/components/ExamTicket";

// 2. إضافة الحالة
const [selectedItem, setSelectedItem] = useState<any>(null);
const [ticketOpen, setTicketOpen] = useState(false);

// 3. استبدال printTicket
const printTicket = (item: any) => {
  setSelectedItem(item);
  setTicketOpen(true);
};

// 4. إضافة المكون
<ExamTicket 
  item={selectedItem}
  isOpen={ticketOpen}
  onClose={() => {
    setTicketOpen(false);
    setSelectedItem(null);
    fetchWorklist(); // تحديث البيانات
  }}
/>
```

## 🎯 المميزات الرئيسية

### ✨ 1. واجهة شاملة
```
┌─ معلومات المريض
├─ معلومات الفحص
├─ معلومات الدفع (مع أزرار تحديث)
└─ ملاحظات
```

### 💾 2. جلب البيانات التلقائي
```javascript
// عند فتح التذكرة
fetchBillingData() {
  // جلب سجل الدفع من DB
  // عرض الحالة والمبلغ
  // تحديث الحقول
}
```

### 🔄 3. تحديث فوري للدفع
```javascript
// عند النقر على زر التحديث
handlePaymentStatusChange("paid") {
  // تحديث قاعدة البيانات
  // تسجيل وقت الدفع
  // عرض رسالة نجاح
  // تحديث الواجهة
}
```

### 📊 4. بيانات من قاعدة البيانات
```sql
-- جدول billing يحتوي على:
SELECT 
  id,
  amount_egp,        -- المبلغ
  status,            -- مدفوع/معلق
  paid_at,           -- تاريخ الدفع
  notes              -- ملاحظات
FROM billing
WHERE exam_order_id = @exam_id
```
