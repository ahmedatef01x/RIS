# 🎨 نظام إدارة العلامة التجارية والصور

## ✅ ما تم إنجازه

تم بنجاح تطوير نظام شامل لإدارة العلامة التجارية والصور للمركز يتضمن:

### 1. الواجهة الأمامية (Frontend)
✅ **صفحة إدارة العلامة التجارية** (`BrandingSettings.tsx`)
- عرض معلومات المركز الحالية
- تعديل البيانات (اسم المركز، العنوان، الهاتف، إلخ)
- إدارة الألوان المخصصة
- معرض صور متقدم
- رفع صور جديدة
- حذف الصور
- واجهة Dialogs سهلة الاستخدام

✅ **التكامل مع النظام**
- إضافة Route جديد في `App.tsx`
- إضافة عنصر في Sidebar مع أيقونة Palette
- حماية الصفحة بـ ProtectedRoute (Admin فقط)

### 2. الخادم (Backend)
✅ **API Routes** (`branding.js`)
- GET /api/branding - جلب البيانات
- PUT /api/branding - تحديث البيانات
- GET /api/media - جلب الصور
- POST /api/media - رفع صورة
- DELETE /api/media/:id - حذف صورة

✅ **التكامل مع الخادم**
- تسجيل الـ Routes في `index.js`
- مصادقة بـ JWT Token
- معالجة الأخطاء والتحقق من الصلاحيات

### 3. قاعدة البيانات (Database)
✅ **الجداول الجديدة**
- جدول `branding` - يخزن معلومات المركز والألوان والشعارات
- جدول `media_gallery` - يخزن الصور والفئات والبيانات الوصفية

✅ **البيانات الافتراضية**
- تم إدراج سجل افتراضي في جدول branding
- جاهزة لاستقبال الصور الأولى

### 4. الميزات الرئيسية

📋 **إدارة المعلومات**
- اسم المركز (عربي/إنجليزي)
- العنوان والهاتف والبريد الإلكتروني والموقع
- نصوص إضافية (التذييل والعن)
- ألوان مخصصة (اختيار بصري)

🖼️ **معرض الصور**
- رفع صور متعددة الصيغ
- تصنيف الصور (أجهزة، منشأة، فريق، أخرى)
- عنوان ووصف لكل صورة
- حذف سهل
- معاينة مباشرة

🔒 **الأمان**
- حماية كاملة بـ JWT
- دقة الأذونات (Admin فقط)
- معالجة أخطاء شاملة

## 🚀 الخطوات التالية

إذا أردت تحسينات إضافية:
1. رفع الصور إلى مجلد خاص بدلاً من Base64
2. ضغط الصور التلقائي
3. توليد صور مصغرة (Thumbnails)
4. إعادة ترتيب الصور بالسحب والإفلات
5. عرض الصور المستخدمة في أماكن مختلفة من الموقع

## 💡 الاستخدام السريع

### تحديث بيانات المركز:
1. انقر على "تعديل" في قسم المركز
2. عدّل البيانات المطلوبة
3. انقر "حفظ التغييرات"

### رفع صورة:
1. انقر على "رفع صورة"
2. أدخل العنوان والفئة
3. اختر الصورة
4. انقر "رفع"

### حذف صورة:
1. مرر الماوس على الصورة
2. انقر على أيقونة الحذف
