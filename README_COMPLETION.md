## 🎨 الميزات المرئية

### ✨ عمود حالة الدفع
```
في جدول Worklist:
✓ يعرض: "مدفوع ✓" أو "معلق ⏳"
✓ يعرض: المبلغ بالـ EGP
✓ يتحدث: عند تحديث الدفع
```

### ✨ نافذة التذكرة
```
تفتح عند النقر "طباعة تذكرة":
✓ معلومات المريض (الاسم، الرقم)
✓ معلومات الفحص (النوع، الجهاز، الموعد)
✓ حالة الدفع والمبلغ
✓ أزرار: تحديث مدفوع / معلق
✓ حقل ملاحظات
✓ زر: طباعة
```

### ✨ التذكرة الطباعية
```
تذكرة احترافية تتضمن:
✓ شعار المركز
✓ بيانات المريض
✓ بيانات الفحص
✓ رقم الزيارة
✓ حالة الدفع والمبلغ
✓ ملاحظات (إن وجدت)
✓ تنسيق عربي RTL
```

## 🚀 كيفية الاستخدام

### الخطوة 1️⃣: تشغيل الخوادم
```bash
# في Terminal 1
cd local-backend
npm run dev

# في Terminal 2
npm run dev
```

### الخطوة 2️⃣: إضافة البيانات (اختياري)
```bash
cd local-backend
node seed-test-exams.js
```

### الخطوة 3️⃣: فتح الموقع
```
http://localhost:5173 → قائمة العمل
```

### الخطوة 4️⃣: استخدام التذكرة
```
1. ابحث عن فحص
2. اضغط الخيارات (⋯)
3. اختر "طباعة تذكرة"
4. ستفتح نافذة التذكرة
```

### الخطوة 5️⃣: تحديث الدفع (اختياري)
```
في النافذة:
• اضغط "تحديث: مدفوع" إذا دفع المريض
• اضغط "تحديث: معلق" إذا لم يدفع
```

### الخطوة 6️⃣: الطباعة
```
اضغط "🖨️ طباعة التذكرة"
← ستُطبع التذكرة مباشرة
```
