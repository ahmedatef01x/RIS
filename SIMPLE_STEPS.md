# 🎬 خطوات بسيطة جداً - استخدام التذكرة

## 1️⃣ افتح الموقع

```
http://localhost:5173
```

---

## 2️⃣ اذهب لـ Worklist

```
القائمة الجانبية ← Worklist (قائمة العمل)
```

---

## 3️⃣ ابحث عن فحص

```
في شريط البحث: اكتب اسم المريض أو رقمه
```

---

## 4️⃣ اضغط على الخيارات

```
النقر على (⋯) بجانب الفحص
```

---

## 5️⃣ اختر "طباعة تذكرة"

```
من القائمة المنسدلة ← طباعة تذكرة
```

---

## 6️⃣ ستفتح نافذة التذكرة

في النافذة تجد:
- ✅ معلومات المريض
- ✅ معلومات الفحص
- ✅ حالة الدفع والمبلغ
- ✅ أزرار للتحديث
- ✅ زر الطباعة

---

## 7️⃣ (اختياري) حدّث حالة الدفع

```
إذا دفع المريض:
  اضغط [✓ تحديث: مدفوع]

إذا لم يدفع:
  اضغط [⏳ تحديث: معلق]
```

---

## 8️⃣ اطبع التذكرة

```
اضغط [🖨️ طباعة التذكرة]
↓
ستُطبع مباشرة على الطابعة
```

---

## 9️⃣ أغلق النافذة

```
اضغط [إغلاق]
↓
الجدول سيُحدّث تلقائياً
```

---

## ✅ انتهيت!
